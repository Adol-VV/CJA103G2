# Changelog

All notable changes to the Momento project will be documented in this file.

## [Unreleased]

### Added
- MockDB integration for three-way data flow (Organizer → Admin → User)
- Dynamic event loading on homepage
- Comprehensive form validation
- Loading states for async operations
- Mobile-optimized responsive design

### Changed
- Refactored organizer dashboard logic into modules
- Standardized panel ID naming convention
- Improved error handling and user feedback

### Fixed
- Syntax errors in organizer dashboard logic
- Script loading order issues
- Duplicate code in product management

## [0.9.0] - 2026-01-10

### Added
- Complete user dashboard (overview, tickets, orders, favorites, settings)
- Organizer dashboard (event/product management, orders, reports, notifications)
- Admin dashboard (approval system, analytics, member management)
- Shopping cart and checkout flow
- Three-role authentication system (Member, Organizer, Admin)
- Responsive design for mobile, tablet, and desktop
- Toast notification system
- QR code ticket scanner

### Technical
- Bootstrap 5 framework
- jQuery for DOM manipulation
- Chart.js for data visualization
- LocalStorage-based MockDB
- Modular JavaScript architecture
- Partial-based HTML structure
